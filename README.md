# Student Attendance System

A Python-based CLI Student Attendance Management System built using Object-Oriented Programming principles.

## Features
- Add new students
- Mark attendance by date
- View attendance report
- Prevent duplicate entries
- Input validation

## Technologies Used
- Python
- OOP
- Dictionary-based storage

## Future Improvements
- JSON-based data persistence
- Attendance percentage calculation
- GUI version
- Database integration
